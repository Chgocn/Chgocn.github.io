Android5.x新特性之 Toolbar和Theme的使用

Android5.x新特性之 Toolbar和Theme的使用

Android5.x新特性之 Toolbar和Theme的使用:http://www.2cto.com/kf/201506/407276.html

1.colorPrimary: Toolbar导航栏的底色。
2.colorPrimaryDark：状态栏的底色，注意这里只支持Android5.0以上的手机。
3.textColorPrimary：整个当前Activity的字体的默认颜色。
4.android:windowBackground：当前Activity的窗体颜色。
5.colorAccent：CheckBox，RadioButton，SwitchCompat等控件的点击选中颜色
6.colorControlNormal：CheckBox，RadioButton，SwitchCompat等默认状态的颜色。
7.colorButtonNormal：默认状态下Button按钮的颜色。
8.editTextColor：默认EditView输入框字体的颜色。

使用方法：请结合：http://www.materialpalette.com/ 下载XML文件进行配置。

LinearLayout divider的用法

LinearLayout divider的用法

use this for horizontal divider

<View
    android:layout_width="1dp"
    android:layout_height="match_parent"
    android:background="@color/honeycombish_blue" />

and this for vertical divider

<View
    android:layout_width="match_parent"
    android:layout_height="1dp"
    android:background="@color/honeycombish_blue" />

OR if you can use the LinearLayout divider, for horizontal divider

<?xml version="1.0" encoding="utf-8"?>
<shape xmlns:android="http://schemas.android.com/apk/res/android" >
    <size android:height="1dp"/>
    <solid android:color="#f6f6f6"/>
</shape>

and in LinearLayout

<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:divider="@drawable/divider"
    android:orientation="vertical"
    android:showDividers="middle" >

If you want to user vertical divider then in place of android:height=”1dp” in shape use android:width=”1dp”

来源： http://stackoverflow.com/questions/15128652/how-to-add-vertical-divider-to-a-horizontal-linearlayout

android:showDividers属性可以设置如下4个值：
none：不显示分隔线；
beginning：在LinearLayout的开始处显示分隔线；
end：在Linearlayout的结尾处显示分隔线；
middle：在LinearLayout中的每两个组件间显示分隔线：

SVN图标无法正常显示

SVN图标无法正常显示

根本原因：Windows Explorer Shell 支持的 Overlay Icon 最多 15 个，Windows 内部就是按图标名称的字母顺序来优先显示的，我们可以调整优先级。
解决办法：修改注册表项
1、删除其他软件。
2、修改优先级。

PagerSlidingTabStrip+ViewPager实现二级导签

PagerSlidingTabStrip+ViewPager实现二级导签

效果：

实现的功能：点击一级导航，会自动切换到二级导航，滑动ViewPager,如果二级导航到最后一个数据，会自动切换到一级导航。
PagerSlidingTabStrip github下载地址：https://github.com/astuetz/PagerSlidingTabStrip
PagerSlidingTabStrip 源码级分析地址：http://a.codekk.com/detail/Android/ayyb1988/PagerSlidingTabStrip%20%E6%BA%90%E7%A0%81%E8%A7%A3%E6%9E%90
思路：Activity中使用PagerSlidingTabStrip与ViewPager绑定，实现一级导航。其次在ViewPager中加载Fragment，最后在Fragment中再次使用PagerSlidingTabStrip与ViewPager绑定，至此结束。
Activity关键布局：

<com.xxx.view.PagerSlidingTabStrip
    android:id="@+id/tabs"
    android:layout_width="wrap_content"
    android:layout_height="match_parent"
    android:layout_centerVertical="true"
    android:layout_toLeftOf="@+id/layout_search"
    android:textColor="@color/text"
    android:textStyle="normal"                   
    app:pstsDividerColor="#00000000"
    app:pstsIndicatorColor="#fff99d1a"
    android:layout_alignParentLeft="true"
    android:layout_alignParentStart="true"
    app:pstsTextAllCaps="false"
    app:pstsUnderlineColor="#00000000"/>

<com.xxx.view.ViewPager
    android:id="@+id/vp_photo"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:layout_below="@id/tabs"/>

Activity中onCreate():

mTabs = (PagerSlidingTabStrip) findViewById(R.id.tabs);
mPhotoGridPager = (ViewPager) findViewById(R.id.vp_photo);
//load Fragment
mFragmentAdapter = new PhotoTabAdapter(this,imgsIndex.getTypes());
mPhotoGridPager.setAdapter(mFragmentAdapter);
mTabs.setViewPager(mPhotoGridPager);
mTabs.setSelectedTextColor(Color.BLACK);

Fragment中布局与Activity类似，故在此省略。
Fragment中onCreateView():

mSubTabs = (PagerSlidingTabStrip) view.findViewById(R.id.tabs_sub);
mPhotoGrid = (PhotoViewPager) view.findViewById(R.id.vp_photo_tab);
// init sub tabs
mSubTabsAdapter = new PhotoGalleryPagerAdapter(getChildFragmentManager(),type);
mPhotoGrid.setAdapter(mSubTabsAdapter);
mSubTabs.setViewPager(mPhotoGrid);

注意：mSubTabsAdapter = new PhotoGalleryPagerAdapter(getChildFragmentManager(),type);此处必须使用getChildFragmentManager().具体参考：http://blog.csdn.net/mybook1122/article/details/24003343
关于二级导航的样式（即选中的tab文字高亮）请参考：http://blog.csdn.net/guolin_blog/article/details/26365683
