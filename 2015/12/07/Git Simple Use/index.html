Git Local 基本操作

初始化

  • 设置全局变量
    git config —global user.name “yourname”
    git config —global user.mail “yourmail@mail.com”
  • 初始化新版本库
    git init
  • 设置忽略的文件
    创建.gitignore文本文件，然后配置需要忽略的文件，可使用正则表达式，最后添加到版本库，进行提交。
  • 添加新文件到版本库
    git add .//添加所有的文件
    git add readme.md//添加readme.md
  • 提交
    git commit -m “add new file”//-m是参数message,引号内的add new file是提交的message的具体值。
