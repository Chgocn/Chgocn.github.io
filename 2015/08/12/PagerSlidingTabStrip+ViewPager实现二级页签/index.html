PagerSlidingTabStrip+ViewPager实现二级导签

效果：

实现的功能：点击一级导航，会自动切换到二级导航，滑动ViewPager,如果二级导航到最后一个数据，会自动切换到一级导航。
PagerSlidingTabStrip github下载地址：https://github.com/astuetz/PagerSlidingTabStrip
PagerSlidingTabStrip 源码级分析地址：http://a.codekk.com/detail/Android/ayyb1988/PagerSlidingTabStrip%20%E6%BA%90%E7%A0%81%E8%A7%A3%E6%9E%90
思路：Activity中使用PagerSlidingTabStrip与ViewPager绑定，实现一级导航。其次在ViewPager中加载Fragment，最后在Fragment中再次使用PagerSlidingTabStrip与ViewPager绑定，至此结束。
Activity关键布局：

<com.xxx.view.PagerSlidingTabStrip
    android:id="@+id/tabs"
    android:layout_width="wrap_content"
    android:layout_height="match_parent"
    android:layout_centerVertical="true"
    android:layout_toLeftOf="@+id/layout_search"
    android:textColor="@color/text"
    android:textStyle="normal"                   
    app:pstsDividerColor="#00000000"
    app:pstsIndicatorColor="#fff99d1a"
    android:layout_alignParentLeft="true"
    android:layout_alignParentStart="true"
    app:pstsTextAllCaps="false"
    app:pstsUnderlineColor="#00000000"/>

<com.xxx.view.ViewPager
    android:id="@+id/vp_photo"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:layout_below="@id/tabs"/>

Activity中onCreate():

mTabs = (PagerSlidingTabStrip) findViewById(R.id.tabs);
mPhotoGridPager = (ViewPager) findViewById(R.id.vp_photo);
//load Fragment
mFragmentAdapter = new PhotoTabAdapter(this,imgsIndex.getTypes());
mPhotoGridPager.setAdapter(mFragmentAdapter);
mTabs.setViewPager(mPhotoGridPager);
mTabs.setSelectedTextColor(Color.BLACK);

Fragment中布局与Activity类似，故在此省略。
Fragment中onCreateView():

mSubTabs = (PagerSlidingTabStrip) view.findViewById(R.id.tabs_sub);
mPhotoGrid = (PhotoViewPager) view.findViewById(R.id.vp_photo_tab);
// init sub tabs
mSubTabsAdapter = new PhotoGalleryPagerAdapter(getChildFragmentManager(),type);
mPhotoGrid.setAdapter(mSubTabsAdapter);
mSubTabs.setViewPager(mPhotoGrid);

注意：mSubTabsAdapter = new PhotoGalleryPagerAdapter(getChildFragmentManager(),type);此处必须使用getChildFragmentManager().具体参考：http://blog.csdn.net/mybook1122/article/details/24003343
关于二级导航的样式（即选中的tab文字高亮）请参考：http://blog.csdn.net/guolin_blog/article/details/26365683
